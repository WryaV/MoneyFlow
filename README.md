# MoneyFlow
MoneyFlow is a command-line app for tracking income, expenses, and managing financial data securely.
## Project Overview
**MoneyFlow** is a simple command-line application designed to help users track their financial income and expenses. The application allows users to add, edit, and remove financial transactions while associating them with specific months. All financial data is saved in a CSV file and can be restored when the program is restarted. The project demonstrates key programming principles like file handling, user authentication, and basic financial calculations.

## Features
- **User Authentication**: Secure access to the application using a password file.
- **Add Transactions**: Users can add new income and expense transactions.
- **Edit/Delete Transactions**: Users can edit or delete existing transactions.
- **Financial Summary**: Calculate and display the total financial flow (Income - Expense - Tax).
- **Persistence**: Data is stored in a CSV file, allowing users to restore their financial data when restarting the application.
